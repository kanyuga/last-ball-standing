How to Play

  1. Select a ball (colour-filled circle) and an empty slot (grey lifeless ring) to jump to.
  2. In the process you must jump over a single ball.
  3. The ball that has been jumped over disappears like magic (Disclaimer: not real magic).
  4. The aim of the game is to only have the last ball standing.
